Git Basics

git init

Initializes a new Git repository in the current directory.

git init

git add

Adds files to the staging area.

git add filename    # Add a specific file
    git add .          # Add all files

git commit

Records changes to the repository.

git commit -m "Commit message"

git status

Shows the status of files in the working directory and staging area.

git status

git log

Shows commit history.

git log
    git log --oneline    # Compact view
    git log --graph      # Show branch structure

Branching & Merging

Branches allow you to develop features, fix bugs, or experiment with new ideas in isolation without affecting the main codebase. Merging combines changes from different branches.

git branch

List, create, or delete branches.

git branch              # List branches
    git branch feature-name  # Create new branch
    git branch -d branch-name # Delete branch

git checkout

Switch branches or restore working tree files.

git checkout branch-name           # Switch to branch
    git checkout -b new-branch        # Create and checkout new branch
    git checkout -                    # Switch to previous branch

git switch

Switch branches (modern alternative to checkout).

git switch branch-name           # Switch to branch
    git switch -c new-branch        # Create and switch to new branch

git merge

Join two or more development histories together.

git merge branch-name        # Merge specified branch into current branch
    git merge --no-ff branch-name # Create merge commit even if fast-forward is possible

Resolving Merge Conflicts

When Git can't automatically merge branches, you need to resolve conflicts manually.

# After a conflict:
    # 1. Edit the conflicted files
    # 2. Mark as resolved
    git add conflicted-file
    
    # 3. Complete the merge
    git merge --continue
    
    # Or abort the merge
    git merge --abort

Remote Repositories

Remote repositories allow you to collaborate with others, backup your work, and synchronize changes across multiple machines. GitHub, GitLab, and Bitbucket are popular Git hosting services.

git remote

Manage remote repositories.

git remote                     # List remotes
    git remote -v                  # List remotes with URLs
    git remote add name url        # Add new remote
    git remote remove name         # Remove remote

git clone

Create a copy of a remote repository.

git clone repository-url                   # Clone repository
    git clone repository-url folder-name      # Clone into specific directory
    git clone --branch branch-name repo-url   # Clone specific branch

git push

Upload local repository content to a remote repository.

git push origin branch-name          # Push branch to remote
    git push -u origin branch-name       # Push and set upstream
    git push --force origin branch-name  # Force push (use carefully!)

git pull

Fetch from and integrate with another repository or branch.

git pull                        # Pull changes from upstream
    git pull origin branch-name     # Pull from specific remote branch
    git pull --rebase               # Pull and rebase local changes

git fetch

Download objects and refs from another repository without merging.

git fetch                       # Fetch all remotes
    git fetch origin               # Fetch from origin
    git fetch --all                # Fetch all branches from all remotes

Advanced Topics

git rebase

Reapply commits on top of another base tip.

git rebase branch-name            # Rebase current branch onto branch-name
    git rebase -i HEAD~3              # Interactive rebase for last 3 commits
    git rebase --continue             # Continue after resolving conflicts

git cherry-pick

Apply the changes introduced by existing commits.

git cherry-pick commit-hash        # Apply specific commit to current branch
    git cherry-pick commit1..commit2  # Apply a range of commits

git stash

Stash changes in a dirty working directory.

git stash                   # Stash changes
    git stash list               # List stashes
    git stash apply              # Apply most recent stash
    git stash apply stash@{n}    # Apply specific stash
    git stash pop                # Apply and remove stash
    git stash drop               # Delete most recent stash

git tag

Create, list, delete, or verify tags.

git tag                          # List tags
    git tag v1.0.0                    # Create lightweight tag
    git tag -a v1.0.0 -m "Version 1.0.0"   # Create annotated tag
    git push origin --tags            # Push tags to remote

git bisect

Use binary search to find the commit that introduced a bug.

git bisect start                # Start bisect session
    git bisect bad                  # Mark current commit as bad
    git bisect good commit-hash     # Mark known good commit
    # Git will checkout commits for you to test
    git bisect good/bad             # Mark current commit as good or bad
    git bisect reset                # End bisect session

git worktree

Manage multiple working trees attached to the same repository.

git worktree add ../path branch-name  # Create new worktree
    git worktree list                     # List worktrees
    git worktree remove ../path           # Remove worktree

Git Cheat Sheet

Creating Repositories

git init Initialize a new repository
git clone url Clone existing repository

Basic Operations

git add file Add file to staging area
git add . Add all changes to staging area
git commit -m "message" Commit staged changes
git status Show repository status
git log Show commit history
git diff Show unstaged changes

Branching & Merging

git branch List branches
git branch name Create branch
git checkout branch Switch to branch
git checkout -b name Create and switch to branch
git merge branch Merge branch into current branch
git branch -d branch Delete branch

Remote Operations

git remote -v List remotes
git remote add name url Add remote
git fetch remote Download from remote
git pull remote branch Fetch and merge from remote
git push remote branch Upload to remote

Undoing Changes

git restore file Discard changes in working directory
git restore --staged file Unstage file
git reset commit Undo commits, keep changes
git reset --hard commit Undo commits, discard changes
git revert commit Create new commit that undoes changes

Advanced Operations

git stash Save changes temporarily
git stash pop Apply and remove stashed changes
git rebase branch Reapply commits on top of branch
git cherry-pick commit Apply commit to current branch
git tag name Create tag at current commit
